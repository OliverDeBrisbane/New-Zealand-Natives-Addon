# NZ Natives Addon
 Minecraft Bedrock behaviour and resource pack that adds various NZ native animals to the game.

- Hector’s / Maui’s Dolphin
- Haast’s eagle
- Hoiho (yellow-eyed penguin)
- Huhu Grub
- Katipo
- Kea
- Kotuku
- Monarch butterfly / caterpillar
- Powelliphanta (giant snail)
- Ruru (Morepork)
- Sea Lion
- Takahe
- Tuatara
- Tui
- Weka
- Weta
